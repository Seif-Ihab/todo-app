# Todo App

A modern, responsive, and feature-rich **Todo App** built with **HTML, Tailwind CSS, and JavaScript**. Manage your tasks efficiently with dark mode support, drag-and-drop reordering, and localStorage persistence.

## Features

- Add, delete, and complete tasks with intuitive UI  
- Filter tasks: **All**, **Active**, **Completed**  
- Clear completed tasks with a single click  
- Drag-and-drop to reorder tasks  
- Light/Dark mode toggle with smooth transitions  
- Persistent data using `localStorage`  
- Responsive design: works on desktop and mobile devices  
- Smooth animations for adding, removing, and completing tasks

## Usage

- **Add a task**: Type in the input field and click "Add"  
- **Complete a task**: Click the checkbox next to the task  
- **Delete a task**: Click the trash icon  
- **Filter tasks**: Use the filter buttons at the bottom of the list  
- **Clear completed tasks**: Click the "Clear Completed" button  
- **Reorder tasks**: Drag and drop tasks to change their order  
- **Toggle theme**: Click the sun/moon icon to switch between light and dark modes
